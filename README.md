# Sales Performance Dashboard (Tableau Project)

## Overview
This project presents an interactive sales performance dashboard built in Tableau.
The dashboard allows users to analyze yearly business performance using key metrics
such as Sales, Profit, and Quantity.

Users can dynamically filter data by year to explore trends and aggregated KPIs.

## Tools & Technologies

- Tableau
- Excel Dataset
- Data Visualization
- Dashboard Design

## Dashboard Features

- Year-wise Sales Analysis
- Profit Trend Visualization
- Quantity Sold Analysis
- Dynamic KPI Cards
- Interactive Year Filter
- Business Performance Insights

## Dashboard Metrics
- Total Sales KPI
- Total Quantity KPI
- Sales by Year
- Profit by Year
- Quantity by Year

  ## Key Insights

- Sales show steady growth across years except a slight drop in year 2021.
- Profit margin also increases every year.
- Quantity sold correlates strongly with overall revenue trends.
